Patitas - Servicio REST

Bienvenido, desde esta página puede navegar y explorar los diversos endpoints que expone el servicio, asi como un pequeña descripcion del mismo



  • Dueños

    Endpoint para manejar Dueños los cuales son los usuarios, desde aqui se realiza el login, registro, etc

  • Mascotas

    Endpoint para manejar Mascotas las cuales estan asociadas con los Dueños, se puden subir imagenes de estas asi como características específicas

  • Adopciones

    Endpoint para manejar publicar Adopciones, se puede especificar la especie del animal, asi como su raza

  • Distritos

    Endpoint para mostrar los Distritos de Lima Metropolitana

  • Especies

    Endpoint para mostrar las Especies de animales domésticos

  • Razas

    Endpoint para mostrar las Razas de las diversas especies que existen
